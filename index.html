O que eu faço

Minhas habilidades abrangem as tecnologias HTML, CSS e JavaScript.

HTML

HTML é a linguagem usada para estruturar o conteúdo de páginas da web, definindo elementos como textos, imagens, links e seções.

CSS

CSS é a linguagem usada para estilizar páginas da web, controlando cores, fontes, layouts e a aparência dos elementos.

JavaScript

JavaScript é a linguagem usada para adicionar interatividade e comportamento dinâmico às páginas da web.
